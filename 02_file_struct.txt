Lostroid 에서는 다음과 같은 규칙을 가집니다.

폴더 : 소대문자 혼합 권장.
파일 : 소문자 권장.
이는 폴더와 파일간의 차이점을 두기 위한 방법임.

참고사항.
[F] : 폴더 표시
- : 상위 폴더
-- : 하위 폴더 또는 파일.

기본  SDK_XBOT 폴더 구조  
- [F] BareMeatal : HAL 이 아닌경우 사용 실제 레지스터 집합체 (옵션)
- [F] Base : Interface 에 관련된 버퍼 및 송 수신 관리 (SPI,UART,SDOI,ADC,DAC,TIME...)
- [F] Cli : Command Line Interfac 기반 통신
- [F] ScheduleManager : 각 Module 의 스케쥴 을 설정 관리 (테이블 관리)
- [F] Bootloader : 부트로더 동작시 필요한 정보 관리
- [F] USB : USB 관련 관리 (CDC)
- [F] Memory : 외부 저장장치 관리 (QSPI,SPI,EMMC)
- [F] Ethernet : 유선 이더넷 칩 관리 (LAN7200)
- [F] Display : 디스플레이 제어 관리 (ST7764)
- [F] Audio : 사운드 및 마이크 칩 제어 관리 (Sound, Mems)
- [F] File_system : 파일시스템 관리 (FAT32, exFAT)
- [F] Camera : 카메라 제어 관리 (OV7670, OV2624)
- [F] Disk : SSD, EMMC, SDCARD 제어 관리 
- [F] DBG : 디버깅 도구
- [F] Wireless : 무선이더넷 칩 관리 (Zigbee, Wifi, Bluetooth, Thread)
- [F] Sensor : 각종 센서 관리 (GPS, IMU, NTC)
- [F] Gpu : 그래픽 가속기 관리  (2D, 3D)
- [F] Codec : 파일 인코딩 디코딩 관리 (bmp. png, doc, H265, jpeg)
- [F] Network : 인터넷 논리 계층 관리( IPV4, IPV6, TCP, UDP, ICMP. ARP)

SDK_XBOT 구조  
- [F] bareMetal (포트 및 통신 속도제어 설정 및 초기화 및 주요 함수 모음 물리적 제어)
